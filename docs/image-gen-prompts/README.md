# 画像生成プロンプト集

リハビリ運動の指導画像を AI 画像生成ツールで作成するためのプロンプトテンプレート集です。

## 目的

自主トレーニング指導箋に使用する運動説明画像を、一貫したスタイルで効率的に生成することを目指しています。

## ファイル構成

| ファイル                             | 説明                                                 |
| ------------------------------------ | ---------------------------------------------------- |
| [model.md](./model.md)               | モデル（人物）を生成するための基本プロンプト         |
| [model.webp](./model.webp)           | 生成されたモデルの参照画像                           |
| [Instructions.md](./Instructions.md) | 運動画像生成時の入力テンプレートと指示               |
| [custom-gem.md](./custom-gem.md)     | カスタムGem（AI エージェント）用のシステムプロンプト |

## 使い方

### 1. モデル画像の準備

`model.md` のプロンプトを使用して、統一されたモデル（人物）画像を生成します。この画像は全ての運動画像で参照画像として使用し、一貫性を保ちます。

### 2. 運動画像の生成

**方法 A: 手動プロンプト入力**

`Instructions.md` のテンプレートに運動の詳細を記入し、画像生成 AI に入力します。

**方法 B: カスタム Gem の利用**

`custom-gem.md` をシステムプロンプトとして設定したカスタム AI エージェントを作成すると、運動の詳細を入力するだけで適切な画像が生成されます。

## 画像仕様

- **形式**: 2 パネル分割（左: 開始姿勢、右: 終了姿勢）
- **背景**: 純白（#FFFFFF）
- **スタイル**: リアルな写真風、スタジオ照明
- **禁止事項**: テキスト、矢印、線、アイコンの描画

## 注意事項

- 生成画像の品質は使用する AI ツールに依存します
- 解剖学的な正確性を確認してから使用してください
- モデルの特徴（服装、髪型など）を維持することで資料全体の統一感を保てます
