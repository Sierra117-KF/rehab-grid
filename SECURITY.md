## サポートされているバージョン

本アプリケーションは静的サイトとしてホスティングされているため、常に最新バージョンが提供されます。セキュリティ修正は最新バージョンにのみ適用されます。

| バージョン | サポート状況 |
| ---------- | ------------ |
| 最新版     | :white_check_mark: サポート対象 |
| 過去版     | :x: サポート対象外 |

---

## 脆弱性の報告

セキュリティに関する脆弱性を発見された場合は、以下のフォームからご報告ください。

**報告窓口（ご意見箱）:**

https://docs.google.com/forms/d/e/1FAIpQLSfCEVpML2K8qPpOaCSapN6BiqUXqSC-jy0ynBK6E0g7eA01TA/viewform

### 報告に含めていただきたい情報

- **脆弱性の概要**: どのような種類の脆弱性か（XSS、CSRF、情報漏洩など）
- **再現手順**: 脆弱性を再現するための具体的な手順
- **影響範囲**: この脆弱性によって起こりうる影響
- **修正案**（任意）: 可能であれば、修正方法のご提案

---

## 対応について

- ご報告いただいた脆弱性は、内容を確認の上、可能な限り速やかに対応いたします
- 対応状況については、必要に応じてご連絡いたします
- 修正が完了するまで、脆弱性の詳細を公開しないようご協力をお願いいたします

---

## お願い

- **公開Issueでの報告はお控えください**: セキュリティ上の理由から、脆弱性の詳細を公開Issueに記載しないでください。悪意のある第三者に悪用される恐れがあります
- **責任ある開示（Responsible Disclosure）へのご協力**: 修正版がリリースされるまで、脆弱性の詳細を非公開としていただけますようお願いいたします

セキュリティ向上にご協力いただける皆様に感謝いたします。

## Supported Versions

This application is hosted as a static site, so the latest version is always provided. Security fixes are only applied to the latest version.

| Version | Supported |
| ------- | --------- |
| Latest  | :white_check_mark: Supported |
| Older   | :x: Not supported |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through the following form.

**Report Form (Suggestion Box):**

https://docs.google.com/forms/d/e/1FAIpQLSfCEVpML2K8qPpOaCSapN6BiqUXqSC-jy0ynBK6E0g7eA01TA/viewform

### Information to Include in Your Report

- **Vulnerability Summary**: Type of vulnerability (XSS, CSRF, information disclosure, etc.)
- **Reproduction Steps**: Specific steps to reproduce the vulnerability
- **Impact**: Potential impact of this vulnerability
- **Suggested Fix** (optional): If possible, your proposed solution

---

## Response Process

- We will review reported vulnerabilities and respond as quickly as possible
- We will contact you regarding the status of the response as needed
- Please refrain from disclosing the details of the vulnerability until the fix is complete

---

## Important Notes

- **Please do not report via public Issues**: For security reasons, please do not post vulnerability details in public Issues. This could be exploited by malicious third parties
- **Responsible Disclosure**: We kindly ask that you keep vulnerability details confidential until a fix has been released

We appreciate everyone who helps improve security.
